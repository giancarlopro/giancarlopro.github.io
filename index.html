Framebot

Biblioteca escrita em C para auxiliar a programação de robôs de chat para o Telegram.

O objetivo da biblioteca é facilitar a comunicação com a API do telegram de criação de bots

Projeito feito em tempo vago com o objetivo de aprender conceitos mais avançados da linguagem C

  • Linguagem: C
  • Criado em: Dezembro de 2016
  • giancarlopro/framebot

Drapion

Biblioteca escrita em Python para auxiliar programadores a utilizarem API's.

Esse biblioteca facilita o uso de qualquer REST API na linguagem python. Sem a necessidade de implementar código específico para a API.

O Objetivo de ter feito esse projeto foi para aprender conceitos de meta-programação em Python

  • Linguagem: Python
  • Criado em: Setembro de 2018
  • giancarlopro/drapion

faker-snippets

O Objetivo de ter feito esse projeto foi por não existir nenhuma extensão equivalente na loja do VSCode.
